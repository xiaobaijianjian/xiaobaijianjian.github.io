JS数据类型及判断总结

JS中有7种内置数据类型，分别是null、undefined、Boolean、Number、string、object、symbol其中，前五种为基本数据类型。object类型又具体包括了function、date、array、object等。

对于类型判断，常用的方法有typeof、instanceof、Object.prototype.toString、constructor

typeof 判断数据类型

number返回number

string返回string

boolean返回boolean

undefined返回undefined

null返回object

function返回function

object返回object

array返回object

NAN返回number

所以，Typeof可以判断除了null以外的基本数据类型，以及function类型、symbol类型、null 会被判断成object

instanceof判断数据类型

检查某对象是否属于某类型,instanceof是通过原型链来检查类型的，所以适用于任何”object”的类型检查

例如：[] instanceof Object 的结果是什么？true

因为[] instanceof Array 、Array instanceof Function、Function instanceof Object
